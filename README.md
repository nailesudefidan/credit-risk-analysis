
# 🏦 Credit Risk Analysis

Bu proje, kredi temerrüt riskini tahmin etmek amacıyla makine öğrenmesi modelleri geliştirmeyi amaçlamaktadır.

## 📊 Veri Seti
- 32,581 kayıt, 12 değişken
- Kaynak: Credit Risk Dataset (Kaggle)

## 🔍 Proje Adımları
- Keşifsel Veri Analizi (EDA)
- İstatistiksel Testler (T-testi, Ki-Kare)
- Veri Ön İşleme & Feature Engineering
- Logistic Regression ve Random Forest Modelleri
- Model Değerlendirme

## 📈 Sonuçlar
| Model | ROC-AUC |
|-------|---------|
| Logistic Regression | 0.86 |
| Random Forest | 0.93 |

## 🛠️ Kullanılan Teknolojiler
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
